Detailed Description

The version class controls the versioning of OpenCK.

Encapsulates application version strings into comparable objects and provides static convenience functions for raw strings.

For comparison purposes, such as for migrating QSettings between versions, or removing deprecated QSettings.

For formatting purposes, such as display strings for window titles.

Checks if a version string is valid.

Checks if a version string is valid. Some examples of version strings:

Valid "1.0.0", "1.0.1a1", "1.0.2b1.dev1", "1.1.dev1", "1.1.12a1", "1.1.12.post1"
